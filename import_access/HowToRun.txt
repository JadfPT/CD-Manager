1. Criar ambiente virtual e instalar deps

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt




2. Copiar o config

copy import_config.example.py import_config.py




3. Preencher import_config.py

Precisa de:

SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY
ACCESS_DB_PATH




4. Testar primeiro em dry-run

python import_access_to_supabase.py




5. Se o preview estiver certo, importar mesmo

python import_access_to_supabase.py --apply
